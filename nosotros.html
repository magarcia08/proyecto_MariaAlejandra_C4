Sobre FakeStore

FakeStore es una tienda virtual desarrollada como proyecto académico con el objetivo de aplicar conocimientos en HTML, CSS y JavaScript.

Nuestra misión

Crear una experiencia de compra clara, sencilla y funcional, permitiendo a los usuarios interactuar con productos reales obtenidos desde una API pública.

Nuestra visión

Convertir este proyecto en una base sólida para el aprendizaje del desarrollo web frontend y la integración de APIs.

Tecnologías usadas

  • HTML5
  • CSS3
  • JavaScript
  • FakeStore API
